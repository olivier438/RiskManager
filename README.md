# RiskManager

## Project Overview
RiskManager is a comprehensive tool designed to help manage and assess risks in various domains. It provides users with the ability to identify, analyze, and mitigate risks, allowing for better decision-making and strategic planning.

## Features
- Risk identification and assessment
- User-friendly interface
- Customizable risk analysis templates
- Integration with other management tools

## Structure

- `index.html` - Page d'accueil et connexion
- `dashboard.html` - Tableau de bord
- `style.css` - Styles de l'application
- `app.js` - Logique principale
- `api.js` - Communication avec l'API
- `config.js` - Configuration
- `auth.js` - Authentification

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/olivier438/RiskManager.git
   ```
2. Navigate to the project directory:
   ```bash
   cd RiskManager
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
   (Make sure you have Node.js and npm installed.)

## Usage
To run the application, use the following command:
```bash
npm start
```

Visit `http://localhost:3000` in your browser to access the application.
